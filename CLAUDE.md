# STAAR Prep Project

STAAR (State of Texas Assessments of Academic Readiness) exam preparation web platform.

## Project Structure

- `index.html` — Main landing page with week navigation
- `week1.html` — Week 1: Diagnostic & Foundations
- `week2.html` — Week 2: Core Skills Mastery
- `week3.html` — Week 3: Advanced Practice
- `week4.html` — Week 4: STAAR Review & Readiness
- `style.css` — Shared stylesheet

## Standards

- Pure HTML/CSS, no build step required
- Keep pages accessible and mobile-friendly
- Follow existing file structure when adding new pages
- Link back to `index.html` on all sub-pages

## Constraints

- No JavaScript frameworks — keep it simple and fast-loading
- All pages must link back to the home index
- Maintain consistent navigation structure across week pages
